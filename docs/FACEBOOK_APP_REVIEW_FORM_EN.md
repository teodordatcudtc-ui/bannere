### 2. Provide instructions for accessing the app

**Answer:**
```
1. Visit: https://socialpilot-ten.vercel.app/
2. Click "Get Started" or "Sign Up" to create a new account
3. Complete the registration form (email and password)
4. Confirm your email address (check your inbox for confirmation link)
5. After confirmation, log in to the application

To test Facebook functionality:
6. Go to "Dashboard" → "Settings"
7. In the "Social Media Accounts" section, click "Connect" for Facebook
8. You will be redirected to Facebook for authorization - authorize the application
9. After authorization, the Facebook account will be connected and appear in the list

To schedule a post:
10. Go to "Dashboard" → "Schedule"
11. Select a generated image (if you don't have one, generate one from "Playground")
12. Fill in the post description
13. Select platforms (Facebook, Instagram, etc.)
14. Select date and time
15. Click "Schedule Post"

Meta APIs Usage Confirmation:
Yes, the application uses Facebook Login and Facebook Graph API through Outstand.so:
- Facebook Login for authentication (permissions: public_profile, email)
- pages_show_list - to list the user's Facebook Pages
- pages_manage_posts - to publish posts on Facebook Pages
- pages_read_engagement - to read engagement statistics
```
